2.3 关键技术决策

Markdown 解析链

  • gray-matter：提取 frontmatter 元数据
  • remark / rehype：Markdown → HTML 转换
  • rehype-highlight：代码块语法高亮
  • remark-toc：自动生成目录（博客）
  • 策略：构建时静态生成 (SSG)，零运行时开销

性能策略

  • ISR：博客/项目页按需增量再生成
  • 图片：next/image 自动 WebP + 懒加载
  • 字体：next/font 零布局偏移
  • 粒子：Canvas 检测 prefers-reduced-motion
  • Bundle：动态 import DemoEmbed（按需加载）
