 Claude Code 生成/修改
        │
        ▼

        │ git push
        ▼

        ▼

主站通过 demo URL 链接跳转 / iframe 嵌入

3.4 子项目接入规范（渐进增强）

Phase 1：链接跳转（立即）

  • 项目卡片 → 外部 Demo 链接（新窗口打开）
  • 项目详情页展示 Markdown 描述 + 截图
  • 零额外成本，每个新项目只需加一个 .md 文件

Phase 2：iframe 嵌入（3+ 项目后）

  • 项目详情页增加"在线体验"按钮
  • 点击展开 DemoEmbed 组件（iframe 全屏/半屏）
  • 子项目需支持 X-Frame-Options 允许嵌入
  • frontmatter 中 demoEmbed: true 控制是否启用

Phase 3：可选增强（未来）

  • 站点内搜索（flexsearch / pagefind）
  • RSS 订阅（博客）
  • Giscus 评论系统（GitHub Discussions）
  • 访客统计（Vercel Analytics）
